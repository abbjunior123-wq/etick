# LP Etick — versão estática para Vercel

Esta pasta é independente do projeto Next.js. Ela contém todo o HTML, CSS,
JavaScript, imagens e vídeo necessários para publicar a landing page.

## Publicar

1. Crie um novo projeto na Vercel.
2. Envie esta pasta como a raiz do projeto.
3. Não é necessário configurar comando de build ou pasta de saída.

O arquivo inicial é `index.html` e o comportamento responsivo já está incluído.
